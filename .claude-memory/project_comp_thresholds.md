These are Aidan's real DMV market numbers as of 2026-05-25. They override any generic comp table. Use them as the *floor* for what counts as a flippable deal — anything priced ABOVE these buy thresholds is not a deal in DMV right now.

**Why:** Aidan called out the scanner for sending him a DRZ125L at $2,200 as a "deal" when he's actually bought the same bike newer/better for $1,200. Generic web ranges were wrong. These are the numbers he's quoted from his own flip history.

**How to apply:** When scoring a listing for any model below, the asking price MUST be at/below the "buy" threshold to score 7+. The "sells" number is the realistic ask, not the dream price. Year/condition adjusts within the band — for the same model, older or beat-up examples need to be cheaper to still flip.

|---|---|---|---|
| Sur Ron LBX | $2,600 | $3,300+ | 2021 |
| DRZ125L | $1,200 | $2,000 | year-dependent |
| CRF250R | $2,000 | $3,000 | year+condition dependent |
| Honda Grom | $1,800 | $2,800 | year-dependent |

**On trucks:** Aidan doesn't restrict to specific models. "Literally any trucks — it just depends on what some random person is letting go for cheap that's what you should be focused on." This is the operating principle: scanner must judge each truck against typical market for that exact year/model, not match a whitelist.

**On 4-wheelers:** Yamaha Raptor 700, Honda TRX450, Yamaha Banshee, YFZ450 all valid. Same principle as trucks — any well-known brand at a clearly-below-market price is a flip.

**THE BIG OPERATING PRINCIPLE (Aidan's own words):**
"I'm not a mastermind so I don't know the exact values for every single vehicle, the goal is for you to learn that." Translation: don't build a static comp database — train the analyzer to estimate market value per listing using its own knowledge of vehicle prices, then flag anything significantly below that estimate. Aidan can't enumerate every model. The scanner has to think generally.
